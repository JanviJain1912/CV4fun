IN THIS WEBSITE OF MACHINE LEARNING AND COMPUTER VISION. STEPS TO RUN THIS PROJECT
1. INSTALL NODE_JS
2. INSTALL XAMPP SERVER
3. INSTALL ALL NODEJS PACKAGE SPECIFIED IN package.json BY USING npm install COMMAND
4. INITIALIZE TABLE IN MYSQL DATABASE
5. START XAMPP SERVER
6. USE node app.js COMMAND TO START APP.JS FILE
7. RUN localhost IN USED PORT.
8. FOR ALL THREE PYTHON MODEL INSTALL REQUIRED LIBERARIES AND SAVE THE CODE FILES WITH .py EXTENSION AND RUN USING file.py COMMAND.
